# Ionfish
## Getting started:
### Prerequisites:
* .net 4.0, 
* directx end-user runtimes (June 2010)
* ruby

### Required ruby gems:
* [Albacore](http://github.com/derickbailey/Albacore) (0.2.0.preview1)
* [nu](http://nu.wikispot.org)

### Installing the .net dependencies:
`rake nu`

### Building the project:
`rake buid`
